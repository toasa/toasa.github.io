ふるさと納税。 佐賀県産のお米がきた。 届いてよかった。

Github pages でブログのデプロイをしたい。N回目の挑戦。 yyyy/mm/dd.md の日記マークダウンファイルから、dd.html を生成するプログラムを作成。 そして、日記のレポジトリに dd.md を push したら、 github.io のレポジトリの CI を動かすよう設定 （トリガ用、 デプロイ用）。 なんか動いてそうな雰囲気：

https://toasa.github.io/diary/

そうそう。こういうのでいいんだよというシンプルさで、結構気に入っている。 Hugoがいいって話も聞くけど、 自分で作れるならそれに越したことはない。

とりあえず今わかってる問題点：

  • マークダウンのテーブル記法がレンダリングできていない。（これとか）
  • スマホで日記をみると、ページのリサイズが行われず見づらい。

https://github.com/toasa/diary/ の方はプライベートリポジトリにして、今後は github pages の方だけ公開するというのもありだよなあ。
