月末を判定するコマンド

この日記はシェルスクリプトでテンプレを生成しているのだが、 月末に歩数を載せたいと思ったので、 月末か否か判定するコマンドを調べた。 軽くネットで調べた限りだと Linux の date コマンドで -d オプションを使えばできることはわかったが、macOS の date コマンドには -d オプションがなく、解説サイトも見当たらない。 mac の man を読むと -v でいけそうとわかった。

仕組みは今日の次の日が１日なら月末と判定するロジック。 Linux と macOS でそれぞれ今日の次の日を取得するコマンドは以下：

Linux$ date -d '1 day'
macOS$ date -v+1d

スクリプト全体は github にある：

https://github.com/toasa/scripts/blob/main/diary.sh
